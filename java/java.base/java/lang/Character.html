How to convert a character to upper case?

                        
//Converts the character argument to uppercase using case mapping information from the UnicodeData file.
//char toUpperCase(char ch)
Character.toUpperCase('a');
//returns: 'A'

How to convert a character to lower case?

                        
Character.toLowerCase('A');
//returns: 'a'

How to the get character representation for a specific digit in the specified radix?

                        
//Determines the character representation for a specific digit in the specified radix.
//char forDigit(int digit, int radix)

Character.forDigit(8,10);
//returns: '8'
Character.forDigit(1,2);
//returns: '1'
Character.forDigit(7,8);
//returns: '7'
Character.forDigit(15,16);
//returns: 'f'
Character.forDigit(10,16);
//returns: 'a'
