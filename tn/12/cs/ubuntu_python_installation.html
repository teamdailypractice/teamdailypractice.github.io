Ubuntu Python Installation Steps

  1. Open terminal. run the below command

    sudo apt-get update

  2. Read the output displayed in the terminal.

    • Are there any errors?
    • Did the command finish successfully?

  3. If no errors, run the below command

    sudo apt-get install idle-python3.6

  4. On success of the above command, type the below command in terminal 

    idle

  5. Python IDE - IDLE - Integrated DeveLopment Environment would open the GUI, where we could write python program.
